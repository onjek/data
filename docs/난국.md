## 개요
**난국**(難國)은 임진왜란을 배경으로 만들어진 [[조선]]의 보드게임이다. 계통상 장기를 원류로 하지만 규칙은 오히려 체스와 비슷하다.

## 역사
난국의 제작자는 전해지지 않는다. 다만 장기를 기반으로 했다는 점은 알려져 있다.

## 구성
### 전지
난국의 게임판은 **전지**(戰地)라고 부른다. 전지는 기본적으로 8×8×4의 삼차원 구조를 가진다. 게임판은 8×8의 바닥 위에 나무토막을 쌓아 올려 만들어지므로 매 대국마다 달라질 수 있다.

### 기물
각 기물의 명칭은 임진왜란 중 도입된 군사 편제인 속오법(束伍法)에서 유래했다.

| 명칭 | 의미 | 설명 |
|---|---|---|
| <ruby><rb>王</rb><rt>왕</rt></ruby> | 왕 | 나라의 임금. |
| <ruby><rb>將</rb><rt>장</rt></ruby> | 영장 | 한 영(營)을 이끄는 장군. |
| <ruby><rb>摠</rb><rt>총</rt></ruby> | 파총 | 한 사(司)를 이끄는 장군. |
| <ruby><rb>砲</rb><rt>포</rt></ruby> | 포수 | 삼수병(三手兵) 중 조총을 다루는 군사. |
| <ruby><rb>馬</rb><rt>마</rt></ruby> | 마병 | 말을 탄 군사. |
| <ruby><rb>射</rb><rt>사</rt></ruby> | 사수 | 삼수병 중 활을 다루는 군사. |
| <ruby><rb>殺</rb><rt>살</rt></ruby> | 살수 | 삼수병 중 창검을 다루는 군사. |
| <ruby><rb>步</rb><rt>보</rt></ruby> | 보병 | 일반적인 군사. |
