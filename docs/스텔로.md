## 개요
> 대한민국 최초의 카페형 인공 언어 포럼, stelo입니다.

**스텔로**는 한국 최대 규모의 인공어 커뮤니티이다.

## 명칭
|---|---|
| # 한국어 | 스텔로 |
| # 에스페란토 | stelo |
| # 영어 | stelo |
| # 일본어 | ステロ |

카페의 명칭 스텔로<small>stelo</small>는 ‘별’을 뜻하는 에스페란토 단어이다.

## 상징
### 로고
스텔로의 현재 로고는 2020년 10월 4일부터 2020년 10월 10일까지 진행된 투표를 통해 채택되었다. 별은 카페의 명칭인 스텔로를, 바벨탑은 인공어 커뮤니티를 뜻한다.
