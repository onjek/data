## 정치 주체
### 정당
**정당**은 정치적 견해를 같이하는 사람듯이 정권을 획득하여 자신들의 정강을 실현하기 위해 조직한 단체이다. 정당은 정책을 추진하고 정치적 책임을 진다. 국민의 의사를 정부 정책으로 연결하는 가장 중요한 정치 조직이므로 현대 정치를 정당 정치라고 부르기도 한다.

#### 정당의 기능
* **정치적 충원**<br>후보자를 공천하고 대표자를 배출함
* **국민 여론 형성·조직화**<br>여론을 형성하고 조직화하여 정부에 전달함
* **정치 사회화**<br>국민에게 정보를 제공해 정치에 관심을 가지게 함
* **정부와 의회 매개**
* **정부 정책 비판·감시**

#### 정당 제도
|  | 일당제 | 양당제 | 다당제 |
|---|---|---|---|
| # 장점 |  | 정치적 책임 소재 분명<br>→ 책임정치<br>정국 비교적 안정 | 정당 선택 범위 넓음<br>→ 국민의 다양한 의사 반영<br>정당간 대립 중재 비교적 쉬움 |
| # 단점 | 독재 정치 나타날 가능성<br>국민의 다양한 의사 반영 難 | 유권자의 정당 선택 범위 좁음<br>과반수 의석을 차지한 정당의 횡포 우려 | 강력한 정책 추진 難<br>정치적 책임 소재 불분명 |

#### 정당을 통한 정치 참여
* **정당 가입**<br>당원으로서 정당 활동에 참여
* **정당이 주최하는 행사·집회 참여**
* **정당의 선거운동 참여**<br>자신의 정치적 견해 밝힘

#### 정당의 문제점
* 소수의 당 지도층이 정당의 의사 결정을 좌우함
* 특정 인물·지역 중심으로 이합집산함

#### 정당 문제점의 해결
* 상향식 의사 결정 구조 확립<br>→ 당내 민주주의 실현
* 정책 중심의 정당으로 변화
