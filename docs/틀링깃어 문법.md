## 일러두기
틀링깃어 문법의 서술에 쓰이는 용어에 대해 통용되는 한국어 번역어를 찾기 어려운 경우, 적당한 단어를 선택해 사용했다. 해당하는 용어들은 아래의 목록에 기재했으므로 엄밀한 표현이 아님을 인지해 두길 바란다.

| 용어 | 번역어 |
|---|---|
| right-here | 초근칭 |
| surroundings | 변칭 |
| alienable | 분리 |
| inalienable | 비분리 |
| alienated | 분리된 |
| relational | 피소유 |
| locative | 위치 |
| proximate | 주칭 |
| obviate | 부칭 |
| partitive | 부분칭 |

## 한정사
| 지칭 | 한정사 |
|---|---|
| # 초근칭 | yá |
| # 근칭 | hé |
| # 중칭 | wé |
| # 원칭 | yú |
| # 변칭 | haa(n) |

**초근칭**은 화자의 위치에 있거나 화자의 소유하에 있는 것을 가리킨다.

**근칭**은 화자의 가까이에 있는 것을 가리킨다.

**중칭**은 화자의 멀리에 있는 것을 가리킨다.

**원칭**은 화자에게서 아주 멀리 있는 것을 가리킨다.

**변칭**은 화자의 주변에 있는 것을 가리킨다.

## 명사
틀링깃어의 명사는 대부분 2음절이다.

### 분류
#### 분리 명사
**분리 명사**(分離名詞, alienable noun)는 소유자 없이 홀로 존재할 수 있는 명사이다. 틀링깃어에서 가장 많은 명사가 속하는 부류이다.

#### 비분리 명사
**비분리 명사**(非分離名詞, inalienable noun)는 소유자 없이 존재할 수 없는 명사이다. 친족어나 신체 부위가 이에 해당한다.

#### 분리된 명사
**분리된 명사**(alienated noun)는 소유자로부터 완전히 분리된 비분리 명사를 가리킨다. 식재료로 쓰이는 동물의 신체 부위가 이에 해당한다.

#### 고유 명사
고유 명사는 일반적으로 접미사를 취하지 않는다. 즉, 소유되거나 복수형이 될 수 없다.

## 대명사
대명사는 인칭과 수로 구분되며, 성별로 구분되지 않는다. 대명사는 일반적으로 조사를 취하지 않는다.

| # 1인칭 | # 단수 || 나 |

| # 2인칭 | # 단수 || 너 |
| ^^ | # 복수 || 너희 |
| # 3인칭 | # 인간 | # 단수 | 그 |
| ^^ | ^^ | # 복수 | 그들 |
| ^^ | # 비인간 || 그것 |
| ^^ | # 주칭 || 바로 그(것) |
| ^^ | # 부칭 || 다른 누군가/무언가 |
| # 4인칭 | # 인간 || 누군가 |
| ^^ | # 비인간 || 무언가 |

**주칭**(主稱, proximate)는 대화에서 이미 언급되었으며 이야기의 주인공으로 취급 받는 인물을 가리킨다.

**부칭**(副稱, obviate)는 대화에서 새로 등장하며 크게 중요하지 않은 인물을 가리킨다.

**4인칭 인간**은 특정되지 않은 사람을 가리킨다.

**4인칭 비인간**은 특정되지 않은 동물, 식물, 사물 등을 가리킨다.

**공간칭**(areal)은 장소나 날씨를 가리킨다. 공간칭은 3인칭 인간 단수 독립 대명사

**재귀칭**은 주어 자신을 가리킨다. 즉, 동작이 스스로에게 일어나는 것을 나타낸다.

**상호칭**은 복수의 주체가 어떤 동작을 함께 행하거나 서로에게 행하는 것을 나타낸다.

## 조사
| 명사 구조 ||||
|:---:|:---:|:---:|:---:|
| # 0 | # 1 | # 2 | # 3 |
| 명사 | 복수/지소 | 피소유 | 격 |

자음으로 끝나는 조사는 항상 고조를 가지며, 모음으로 끝나는 조사는 앞 음절의 반대 성조를 가진다.

모음으로 끝나는 단어에 모음으로 시작하는 조사가 결합하면 사이에 활음이 삽입된다. 평순모음 사이에는 y가, 원순모음 사이에는 w가 삽입된다.

자음으로만 이루어진 조사는 앞 음절에 영향을 주지 않지만, 모음을 포함하는 조사는 종종 앞 음절의 단음을 저조 장음으로 바꾼다.

아래 표에 있는 유기음들은 모음으로 시작하는 조사가 뒤따를 때 무기음화된다.

| 유기음 | 무기음 |
|---|---|
| ch | j |
| t | d |
| tl | dl |
| ts | dz |
| k | g |
| kw | gu |
| ḵ | g̱ |
| ḵw | g̱u |

다음 중 하나로 끝나는 것은 원순음으로 끝난다고 하고, 그렇지 않은 것은 평순음으로 끝난다고 하자.

* w
* 원순모음
* 원순모음 + 연구개음 또는 구개수음

일반적으로 평순음으로 끝나는 단어는 i로 시작하는 형태를, 원순음으로 끝나는 단어는 u로 시작하는 형태를 취한다.

단어 끝에 오는 aa는 조사 앞에서 a로 바뀌며, 이때 조사 또한 저조가 된다.

### 복수/지소
#### -xʼ
명사에 조사 -xʼ가 결합하면 복수형이 된다.

> hít → hít**xʼ**  
> 집 → 집**들**

> té → téi**xʼ**  
> 바위 → 바위**들**

#### -kʼ
명사에 조사 -kʼ가 결합하면 지소형(指小形, diminutive)이 된다. 즉, 원래의 명사보다 더 작은 것을 나타낸다. 명사에 -kʼ가 결합한 형태가 발음하기 어려운 경우에는, -kʼ 대신에 형용사 óoskʼ를 사용할 수 있다.

> ḵóok → ḵóok**kʼ**  
> 상자 → **작은** 상자

> lítaa → lítaa**kʼ**  
> 칼 → **작은** 칼

#### -xʼi sáani
명사에 복수 조사 -xʼi가 결합하고 그것을 지소형 후치 형용사 sáani가 수식하면 복수 지소형이 된다. 복수 지소사는 앞에 오는 명사의 성조를 평탄화시키는 경향이 있다.

> léiḵʼw → léiḵʼw**xʼi** **sáani**  
> 볼락 → **작은** 볼락**들**

> chʼáakʼ → chʼáakʼ**xʼi** **sáami**  
> 독수리 → **작은** 독수리**들**

### 피소유
#### -<sup>i</sup>/<sub>u</sub>
틀링깃어에서는 소유주 대신 피소유주에 속격 조사 -<sup>i</sup>/<sub>u</sub>를 붙인다.

> hít → i híd**i**  
> 집 → 너**의** 집

> lítaa → du líta**yi**  
> 칼 → 그**의** 칼

| # 근접격 | -g̱aa |
| # 향격 | -de |

| # 도착격 | -t |

#### -ch
능격 조사 -ch는 타동사의 주어를 나타낸다.

> X̱aanásʼ awliyéx̱ wé ḵáa.  
> x̱aanásʼ-∅ a-∅-ÿu-li-yéix wé ḵáa  
> 서까래-絶 3單.非人.對-3單.人.主-完望-l類-짓다 中 남자  
> 저 남자가 서까래를 만들었다.

#### -∅
절대격은 표시되지 않는다.

> Tlaagú tlein á.  
> tlaagú-**∅** tlein á  
> 전설-**絶** 큰 3單.非人.獨  
> 그것은 위대한 전설이다.

#### -xʼ ~ -∅
처격 조사 -xʼ ~ -∅는 ‘(장소/상황)에서’, ‘(시간)에’라는 뜻을 나타낸다.

#### -u
독립 처격 조사 -u는 ‘~에 있다’라는 뜻을 나타낸다.

#### -de
향격 조사 -de는 ‘~로’, ‘(기한)까지’, ‘~의 방식으로’라는 뜻을 나타낸다.

#### -t
도착격 조사 -t는 ‘~에 도착하여’라는 뜻을 나타낸다.

#### -náx̱
통과격 조사 -náx̱는 ‘~를 통하여’, ‘~를 따라서’라는 뜻을 나타낸다.

#### -dáx̱ ~ -tx̱ ~ -x̱
탈격 조사 -dáx̱ ~ -tx̱ ~ -x̱는 ‘~로부터’, ‘~ 이래로’라는 뜻을 나타낸다.

#### -x̱
경로격 조사 -x̱는 ‘~를 따라서’라는 뜻을 나타낸다.
