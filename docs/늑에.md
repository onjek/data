## 개요
**늑에**<small>(백어: nikkê)</small>는 특정 생물의 의식 활동에 의해 발생하는 힘으로, 기본 상호작용 중 하나이다.

## 명칭
|---|---|
| #한국어 | 늑에 |
| # 일본어 | ヌエ |
| # 영어 | nikkê |

늑에의 명칭은 ‘영혼’을 뜻하는 [[백어]] 단어 nikkê에서 유래한 것이다. 보고서 [[궛우 지역의 영혼 이용]]에서 최초로 언급되었다.
